overflow do CSS

Controla como o conteúdo deve ser exibido caso ultrapasse os limites do container. É necessário informar altura caso contrário o elemento irá se ajustar automaticamente e o overflow não funcionará

Exemplos:

overflow: visible;

O conteúdo será mostrado mesmo que passe da margens estabelecidas para o elemento.
